# Sample Project

This folder contains a minimal static sample to get started.

Files added:

- `index.html` — sample HTML page
- `css/styles.css` — basic styles
- `js/app.js` — small interactive script

How to use:

1. Open `index.html` in your browser.
2. Click the "Click to greet" button to try the example.

Optional: initialize a Git repo, commit the files:

```powershell
git add .
git commit -m "Add sample project content"
```
